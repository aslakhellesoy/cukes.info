h1. cukes.info

This is the new website for cukes.info. It uses Jekyll.

h2. Software
* Install pygments
* Install http://github.com/aslakhellesoy/cucumber-pygments-lexer : sudo python setup.py install
* Install http://github.com/aslakhellesoy/pygments-groovy : sudo python setup.py install
* Install http://github.com/aslakhellesoy/jekyll : rake install
* Install the latest cucumber gem

h2. Building

The i18n examples are built from Rake:

* Regenerate _includes/i18n_examples.html:
* rake i18n_examples
* jekyll --server --auto
